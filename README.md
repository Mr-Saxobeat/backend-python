# backend-python
Usando o módulo python http.server para criar um servidor backend simples.

Para rodar o script:
  - Instale o virtualenv:
  ``pipenv install``
  - Execute o virtualenv:
  ``pipenv shell``
  - Execute o servidor:
  ``python serv.py``
  - Acesse o endereço no seu navegador:
  ``http://127.0.0.1:8080/``
